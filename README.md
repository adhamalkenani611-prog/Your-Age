عنوان المشروع:
حاسبة العمر الدقيقة (Age Calculator)

الوصف القصير:
موقع ويب تفاعلي لحساب العمر بدقة (سنوات، أشهر، أيام، ساعات) باستخدام HTML/CSS/JavaScript مع ميزات حديثة وتجربة مستخدم ممتازة.

الوصف التفصيلي:

حاسبة العمر الدقيقة هي صفحة ويب عربية متكاملة وعصرية تم تطويرها بواسطة أدهم محمد عبدالسلام الكناني. تسمح للمستخدم بإدخال تاريخ الميلاد (اليوم، الشهر، السنة) وتعرض العمر الفعلي بالسنوات والأشهر والأيام والساعات، مع حسابات دقيقة للتوقيت الفعلي.

الميزات الرئيسية:

واجهة مستخدم أنيقة وعصرية (تصميم متدرج وresponsive).
تحقق فوري من الإدخالات ودعم رسائل خطأ للحقل.
حساب عمرك بالكامل:
سنوات + أشهر + أيام + ساعات.
خيارات إضافية:
حساب العمر بالأيام فقط.
عرض تاريخك بعد عدد معين من السنوات.
وضع ليلي/نهاري مع حفظ الحالة (dark mode).
حفظ النتيجة في المتصفح (localStorage) واستعادة تلقائية.
مشاركة النتيجة (Web Share API ونسخ للحافظة).
زر إعادة ضبط لإعادة تعيين الحقول.
أزرار مخصصة ونظام ألوان احترافي.
Footer يتضمن اسم المطور ومعلومات الاتصال (WhatsApp، إيميل).
